## Projektbeschreibung

Dieses Projekt implementiert einen Stromregelkreis (Stromregelkreis) für ein Feldorientiertes Regelungssystem (FOC). Das Projekt verwendet:

- **Simulink-Modell**: `Stromregelkreis.slx` - Hauptregelkreismodell
- **dSPACE RTI1104**: Real-Time Interface für Hardware-in-the-Loop (HIL) Tests
- **MATLAB R2024b**: Entwicklungsumgebung

Das System wurde für die Echtzeitregelung auf dSPACE Hardware konfiguriert und ermöglicht die Simulation und Validierung des Stromregelkreises unter realen Bedingungen.
